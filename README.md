# 임베디드시스템설계및실험

## 주차별 미션

:two: 환경 설정 및 컴파일, 업로드

:three: 조이스틱 Up, Down, Right, Left 제어 및 LED 점등

:four: 조이스틱 및 스위치로 모터 제어

:five: 버튼으로 USART 통신하여 오실로스코프 및 putty에 결과 출력

:seven: Putty와 Bluetooth 모듈간 통신을 위한 펌웨어 작성

:ten: LCD Touch 동작으로 Timer/LED 제어
